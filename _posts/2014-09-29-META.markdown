>- `metadata` **元数据** 关于数据的信息
>- 规定页面的**描述**、**关键词**、**文档作者**、**最后修改时间**以及其他元数据
>- **元数据**可用于浏览器如何显示内容或者重新加载页面、利于搜索引擎

- generator
- description
- keyword
- author
